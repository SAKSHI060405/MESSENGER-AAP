# 💬 Convo-Z

A modern real-time messaging application built using **Socket.IO**, **Node.js**, **HTML**, **CSS**, and **JavaScript**. Convo-Z enables users to communicate instantly through a clean and responsive chat interface.

---

## 🚀 Features

* Real-time messaging using Socket.IO
* Instant user join and leave notifications
* Responsive and modern UI
* Live message broadcasting
* User identification through custom usernames
* Lightweight and fast communication system

---

## 🛠️ Tech Stack

### Frontend

* HTML5
* CSS3
* JavaScript (ES6)

### Backend

* Node.js
* Socket.IO

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/yourusername/convo-z.git
```

### Navigate to project directory

```bash
cd convo-z
```

### Install dependencies

```bash
cd nodeServer
npm install
```

### Start the Socket.IO server

```bash
nodemon index.js
```

or

```bash
node index.js
```

### Launch Frontend

Open `index.html` using Live Server or any local development server.
